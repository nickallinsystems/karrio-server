# Karrio Server (Custom)

Custom Karrio server image with ShipEngine connector pre-installed.

## What this does

- Extends the official `karrio/server:2026.1.32` image
- Installs `karrio.shipengine==2026.1.32` for ShipStation/ShipEngine integration
- Enables carrier connections to ShipEngine through Karrio's unified API
